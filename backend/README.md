# OPC Backend

FastAPI service for auth, content, knowledge, trend, image, workspace, and model-router APIs.

## Current Sprint

Sprint 1 establishes:

- App factory and health endpoint.
- SQLAlchemy session management.
- Core database models.
- Alembic baseline migration.
- Phone/password auth.
- Workspace dashboard summary.

## API Prefixes

- `/health`
- `/api/auth`
- `/api/workspace`

Upcoming phases add content, knowledge, trends, images, review, and publishing records.
